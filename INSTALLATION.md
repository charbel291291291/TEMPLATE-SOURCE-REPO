# Installation Guide

This template is designed to be easy to install and customize.
No coding knowledge is required.

---

## Requirements
- Node.js 18 or later
- npm (comes with Node.js)

---

## Installation Steps

1. Unzip the downloaded template
2. Open a terminal inside the project folder
3. Install dependencies:
   npm install
4. Customize your website by editing:
   src/config/site.json
5. Start the development server:
   npm run dev

The website will be available at:
http://localhost:5173 (or the port shown in the terminal)

---

## Deployment

You can deploy this template easily using:
- Vercel
- Netlify

Simply connect the project and deploy.

---

## Customization Notes

- All text, images, and contact details are controlled from `site.json`
- No backend setup is required
- WhatsApp booking works automatically
